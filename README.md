# SDL2-GameOfLife
Game of Life written using SDL2 library.

Controls: 
  - LEFT MOUSE CLICK to draw/delete patterns
  - 's' to start/pause
  - 'r' to reset
  - 'c' to clear board
  - 'UP ARROW' to speed up
  - 'DOWN ARROW' to slow down

Compiled with provided Makefile.
